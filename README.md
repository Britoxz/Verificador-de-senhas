# Verificador de Senha Forte

Projeto de segurança da informação para análise de força de senhas.

## Funcionalidades
- Verificação de força da senha
- Análise de caracteres especiais
- Verificação de números e letras maiúsculas
- Feedback instantâneo

## Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript

## Objetivo do Projeto
Aplicar conceitos básicos de cibersegurança e validação de dados.

## Como Executar
1. Baixe os arquivos
2. Abra o index.html no navegador
